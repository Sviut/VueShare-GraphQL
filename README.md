# VueShare-GraphQL


In VueShare-GraphQL folder
1. Install backend packages

- `npm i`
2. Install frontend packages

- `cd client`

- `npm i`

___

Use `npm run dev` in root folder to start project

GraphQl playground - `http://localhost:4000/`

FrontEnd - `http://localhost:8080/`
